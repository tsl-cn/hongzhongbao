# 安全机制文档

## 核心原则

> **牌局必须安全、公平、公正。外部攻击者（包括 AI 模型）不能通过反向破解随机算法预测牌序。**

---

## 洗牌安全（四层防线）

### 第一层：CSPRNG

全部随机操作使用 `crypto.randomInt()` / `crypto.randomBytes()` — Node.js 内核 CSPRNG。

| OS | 底层熵源 |
|---|---|
| Linux | `getrandom()` → 硬件中断 + 磁盘 I/O + 网络包时序 |
| Windows | `BCryptGenRandom()` → 系统熵池 |
| macOS | `getentropy()` → 内核 CSPRNG |

CSPRNG 的安全性基于计算复杂性假设——破解需要 ~2^128 次操作，任何已知计算模型（包括 AI）都无法在有限时间内完成。

### 第二层：256-bit 密钥盐

每局独立生成 32 字节（256-bit）密钥盐：

```
salt = crypto.randomBytes(32) ⊕ process.hrtime().nanoseconds()
```

- 盐存储在 `Wall._salt`（JavaScript 闭包内），**永不序列化、不发送给客户端、不写入日志**
- 盐的 **SHA-256 哈希**记录在游戏日志中，用于事后审计
- 哈希不可逆——攻击者拿到哈希也无法还原盐

**安全保证：** 攻击者要预测牌序，必须同时：
1. 破解 OS 内核 CSPRNG（2^128 计算复杂度）
2. 窃取服务器内存中的密钥盐（需要服务器权限）

两项缺一不可——这是信息论级别的安全，而非仅计算级别。

### 第三层：双重 Fisher-Yates

```
Pass 1: for i = n-1..1:  j = crypto.randomInt(i+1);  swap(arr[i], arr[j])
Pass 2: for i = n-1..1:  j = HMAC-SHA256(salt, i);   swap(arr[i], arr[j])
```

- **第一遍**：CSPRNG —— 外部不可预测
- **第二遍**：HMAC-SHA256(salt, 位置) —— 即使 CSPRNG 被攻破，盐提供独立第二密钥

两层正交：攻击者必须同时攻破两个独立系统。

### 第四层：中局随机洗牌（≥11 次）

起牌后剩余 ~83 张，分 8 段随机取触发点 + 3 个强制里程碑（12/8/4 张）。

每次中局洗牌发生在不同的纳秒时刻，CSPRNG 状态已随时间和操作推进，产生全新的独立随机序列。

**效果：** 即使攻击者奇迹般预测了初始牌序，第一次中局洗牌后所有后续摸牌完全不可追踪。

## AI 反作弊

### 信息防火墙（AiView）

```
GameState ──❌ 防火墙阻断──→ AiView（闭包隔离）──✅ 仅公开信息──→ AiPlayer
```

AI 只能获取：
- 自己手牌、副露、弃牌
- 所有人的公开副露和弃牌堆
- 各家手牌张数（仅数量）
- 牌墙剩余张数（仅数量）

AI 绝对不能获取：
- ❌ 其他玩家手牌内容
- ❌ 牌墙未摸牌面
- ❌ GameState 内部引用

详见 `server/src/ai/ANTI_CHEAT.md`。

## 完整性校验

每次洗牌后执行：
- 牌墙恰好 136 张
- 每种牌面（0-33）恰好 4 张
- 无非法牌值

校验失败 → 立即 `throw Error`，终止游戏（防止内存损坏导致的非公平对局）。

## 审计日志

| 事件 | 日志 | 说明 |
|------|------|------|
| 开局洗牌 | `🔐 洗牌完成 — 盐哈希: a1b2...` | 盐的 SHA-256（不可逆） |
| 中局洗牌 | `🔄 牌墙剩余N张，随机洗牌（第X次）` | 触发点和次数 |
| AI 决策 | `[AI-AUDIT] {"action":"discard",...}` | AI 决策时的公开信息快照 |

审计日志用于事后验证公平性——任何异常决策都可以追溯到具体的信息状态。

## 外部攻击者为什么无法破解

| 攻击方式 | 为什么无效 |
|----------|-----------|
| AI 模型分析多局牌序找规律 | CSPRNG 输出无统计规律，是密码学安全的 |
| 暴力枚举 PRNG 种子 | 种子空间 2^256，宇宙年龄都不够 |
| 从牌序反推 PRNG 状态 | Fisher-Yates 是单向置换，反推等价于破解 CSPRNG |
| 窃听网络包 | 客户端从不接收牌墙数据，只收到自己手牌和公开信息 |
| 服务器入侵获取盐 | 需要 OS 级权限，此时安全性已不在牌局层面 |
