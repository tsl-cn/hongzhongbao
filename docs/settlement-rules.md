# 红中宝自摸麻将 — 结算规则（终版）

> 4层架构，全场和恒为 0

---

## 第1层：非胡家明暗杠结算

明杠 = 1番/杠、暗杠 = 2番/杠（**只算非胡家**）

- 每家从其他 3 家各收取对应番数
- 胡家的明暗杠已在第 2 层手牌番型中计算（通过 FanCalculator）

## 第2层：手牌胡牌结算

`f = FanCalculator(胡家手牌 + 胡家副露 + 胡家明暗杠)`

| 胡牌方式 | 胡家 | 被点炮家 | 其他 2 家 |
|---------|------|---------|----------|
| 自摸（含天胡、人胡） | `+f×3` | — | `-f` |
| 抢杠胡 / 地胡（点炮） | `+f×3` | `-f×3` | `0` |

## 第3层：马牌结算

### 标准自摸

每匹马独立归零：

| 马的类型 | 该匹价值 |
|---------|---------|
| 胡家中马 | `+f × 所有非胡家不中马数` |
| 胡家不中马 | `0` |
| 非胡家中马 | `+f × 其他非胡家不中马数` |
| 非胡家不中马 | `-f × 所有非自己家中马数` |

### 抢杠胡 / 地胡（站队规则）

按马的 `mapped` 分阵营：

| 马的 mapped→ | 拥有者 | 每匹价值 |
|-------------|--------|---------|
| → 胡家 | 胡家自风马 | `+3f × 对方阵营总马数` |
| → 胡家 | 其他人的实马 | `+3f` |
| → 被点炮家 | 被点炮家自风马 | `-3f × 对方阵营总马数` |
| → 被点炮家 | 其他人的实马 | `-3f` |
| → 局外人 | 任意 | `0` |

- **对方阵营** = mapping 到对面的总马数（含自风马 + 实马）
- **"其他人"** 排除对面的当事人

## 第4层：加总

```
玩家净收支 = 层1 + 层2 + 层3
```

全场和为 0 ✅

## 数据结构

每匹马 3 属性：
- `owner` — 谁买的
- `mapped` — 牌面映射到的座位
- `isHit` — `mapped === 胡家`

**自风马** = mapped = 持牌人自己的马
**实马** = 选马阶段额外买的马

**流局** = 各家不输赢
**旁观者买马** = 此后通过追加马 owner 实现，不影响公式
