# 红中宝自摸麻将 — 结算规则（终版）

> 4层架构，全场和恒为 0

---

## 数据结构

每匹马 3 属性：
- `owner` — 谁买的
- `mapped` — 牌面映射到的座位
- `isHit` — `mapped === 当前子层中心人`

**马牌来源：**
- **自风马** — 每人固定 1 匹，`mapped = owner`
- **实马** — 选马阶段额外买的马

每局总马数 = 4（自风马）+ Σ(每人实马数)

---

## 统一子层公式

所有子层（每杠、胡牌）使用同一套公式：

> 中心人 = C，f = 当前子层番值
> hit camp = `mapped=C` 的马，miss camp = `mapped≠C` 的马

| 马类型 | 每匹价值 |
|-------|---------|
| hit, owner=C | `+f × \|miss camp\|` |
| hit, owner=X≠C | `+f × (\|miss camp\| - \|X 在 miss camp 中的马数\|)` |
| miss, owner=X≠C | `-f × (\|hit camp\| - \|X 在 hit camp 中的马数\|)` |
| miss, owner=C | `0` |

单子层恒归 0 ✅

---

## 第1层：非胡家明暗杠（逐杠子层）

每个非胡家的每个杠独立成子层：

- 中心人 = **杠主**
- f = 明杠=1 / 暗杠=2

所有马（自风马 + 实马）全参与。

---

## 第2层：手牌胡牌结算（1个子层）

- f_w = FanCalculator(胡家手牌 + 胡家副露 + 胡家明暗杠)
- 中心人 = **胡家**
- f = f_w

---

## 第3层：加总（每马维度）

```
每马净价值 = Σ(各子层分配的 value)
```

---

## 第4层：各家汇总

```
玩家净收支 = Σ(自家所有马的净价值)
```

全场和为 0 ✅

## 特殊场景

- **流局** = 各家不输赢
- **旁观者买马** = 通过追加马 owner 实现，不影响公式（待定）
- **抢杠胡 / 地胡站队** = 略，后续确认
